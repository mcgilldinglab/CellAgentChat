Quickstart
==========

Typical workflow
----------------

1. Prepare an AnnData object or the legacy input tables.
2. Load the ligand-receptor database.
3. Preprocess expression if you are starting from raw counts.
4. Run the communication workflow with or without spatial distance.
5. Review interaction matrices, significant ligand-receptor pairs, and receiving scores.

Recommended starting points
---------------------------

- Use the legacy modules included in ``src/`` for the ``0.1.0`` workflow.
- Start from the tutorial notebooks in ``tutorial/`` for end-to-end examples.

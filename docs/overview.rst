Overview
========

CellAgentChat integrates gene expression data, ligand-receptor prior knowledge, spatial coordinates, and optional pseudotime information to estimate cell-cell communication.

The framework models each cell as an agent and quantifies interaction strength using three components:

- ligand diffusion rate
- receptor receiving rate
- receptor conversion rate

Key capabilities
----------------

- Estimate cell-cell interactions from spatial transcriptomics coordinates.
- Analyze interactions across pseudotime trajectories.
- Identify relevant ligand-receptor pairs between cell populations.
- Compute per-cell receiving scores.
- Explore in-silico receptor perturbations.
- Study short- and long-range communication behavior.
- Run dynamic simulations and animation-based inspection workflows.

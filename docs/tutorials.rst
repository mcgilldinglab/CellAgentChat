Tutorials
=========

The repository includes notebook-based tutorials under ``tutorial/``:

- ``Tutorial.ipynb`` for the main non-pseudotime workflow
- ``Tutorial_with_imputation.ipynb`` for expression-imputed analysis
- ``Tutorial_with_spot_deconvolution.ipynb`` for deconvolved spatial workflows

Suggested reading order
-----------------------

1. Start with the main tutorial.
2. Review imputation only if your preprocessing workflow requires it.
3. Review spot deconvolution if your data originate from spot-based platforms.
